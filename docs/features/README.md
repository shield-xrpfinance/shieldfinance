# Features Overview

Shield Yield Vaults provides a comprehensive suite of features for XRP liquid staking:

## Core Features

### [Wallet Integration](wallet-integration.md)
Secure connection to multiple wallet providers with real-time balance tracking.

### [Vault Management](vault-management.md)
Browse and interact with multi-asset vaults across different risk tiers.

### [Deposits & Withdrawals](deposits-withdrawals.md)
Seamless on-chain transactions with Xaman QR code signing.

### [Transaction Signing](transaction-signing.md)
Secure blockchain transaction signing with Xaman integration.

### [Portfolio Tracking](portfolio-tracking.md)
Real-time monitoring of positions, rewards, and transaction history.

### [Token Swapping](swap.md)
Instant trading of $SHIELD tokens with FLR, wFLR, and USDT via SparkDEX V3 integration.

## Additional Features

- **Network Switching**: Toggle between mainnet and testnet
- **Auto-Refresh**: 30-second wallet balance updates
- **APY Tracking**: Historical performance charts
- **Multi-Asset Support**: XRP, RLUSD, and USDC
- **Risk Tiers**: Conservative, Moderate, and Aggressive vaults
